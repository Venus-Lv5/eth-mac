- `docs/README.md`: index + cách dùng
- `docs/requirements.md`: yêu cầu (RQ-xxx)

- `docs/register-map.md`: register/bitfield
- `docs/verification-plan.md`: kế hoạch verify
- `docs/traceability.md`: truy vết RQ ↔ RTL ↔ test
- `docs/changelog.md`: decision log
